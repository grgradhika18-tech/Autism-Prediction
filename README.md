Autism Detection System (AI/ML Project)

## 📌 Overview
This project focuses on detecting Autism Spectrum Disorder (ASD) using Machine Learning and Deep Learning techniques.  
It combines **screening data** and **image-based analysis** to build predictive models that assist in early diagnosis.

---

## 🎯 Objectives
- Predict autism likelihood using screening questionnaire data
- Classify autism from facial/image data using deep learning models
- Explore multiple ML/DL techniques for better accuracy
- Provide a scalable AI-based healthcare solution

---

## 🛠️ Tech Stack
- Python 🐍
- TensorFlow / Keras
- Scikit-learn
- NumPy & Pandas
- Matplotlib / Seaborn
- OpenCV (for image processing)

---

## 📊 Datasets

### 🔹 Screening Dataset
- https://www.kaggle.com/datasets/fabdelja/autism-screening-for-toddlers  
- https://www.kaggle.com/datasets/uppulurimadhuri/dataset  

These datasets contain questionnaire-based features used for autism screening.

---

### 🖼️ Image Dataset
- https://www.kaggle.com/datasets/radhgarg/autism-image-dataset  

This dataset is used for image-based autism classification using CNN models.
